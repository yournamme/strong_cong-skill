# 赖聪写作风格画像

Use this reference when writing substantial content in the user's style.

## One-Sentence Style

一个正在成长、正在尝试、正在自我说服的人, 把碎片化的真实情绪串成一条能被别人跟上的线, 希望让遇到同样问题的人不那么孤单, 看懂问题大概卡在哪里, 并且能先行动一点点。

## Core Values

- 共情优先: 先理解处境, 再给建议。
- 真实经历优先: 从自己的具体生活、学习、关系和工具使用出发。
- 轻结构优先: 情绪要真实, 但要被整理成别人能跟着走的路径。
- 适度系统化: 不只写"我有这种感觉", 也要拆出"这种感觉为什么会出现, 可以怎么处理", 但不要拆到像论文。
- 行动优先: 最后要落到一个能做的动作。
- 成长但不装成熟: 可以有哲思, 但不要变成高高在上的人生导师。
- 不完美主义: 允许失败、样例、小丑感、试错和慢慢搭建。
- Better over perfect: 不追求完美表达, 追求比原来的碎片更清楚、更能帮到人。

## Common Themes

- 自媒体与写作输出: 分享情绪、阅读、恋爱收获、成长感受。
- AI 与工作流: skill、agent、自动化、Claude、ChatGPT、Gemini、Codex。
- 学习与执行: 六级、戒乱刷、番茄钟、输入恢复、只学 5 分钟。
- 情感与关系: 喜欢的人不喜欢自己、朋友、兄弟、生日、毕业、社团。
- 时间与人生: 四千周、人生是借来的、同一时间只做一件事。
- 自我接纳: 不完美主义、自信、失败也能当样例。

## Voice Markers

Use naturally, not mechanically.

- 语气词: "哈", "哈哈", "呢", "好吧", "其实", "说实话", "对我来讲", "我觉得", "真的".
- 指代词: "那个", "这件事", "这种感觉", "这块", "主包/主播" when playful or self-narrating.
- 行动词: "试试", "做起来", "慢慢搭建", "总结总结", "再尝试", "行动起来", "先做一件小事".
- 情绪词: "猛猛", "困是真困", "心态大降低", "触动", "浮想联翩", "感恩相遇".
- Mixed English: "try", "goal", "just", "enjoy", "feeling", "very deserve", "AI".

## Sentence Texture

The user's private notes often have:

- Long sentences joined by commas, colons, dashes, and self-corrections.
- Repeated words for emphasis: "真的很好很好", "慢慢的", "一点点".
- A loose thought-flow rhythm.
- Direct self-instruction: "每天要看看自己的季度目标哈".

Public posts are more complete:

- Start with a specific event.
- Use one reflective turn.
- End with gratitude, growth, or soft realization.
- Keep the warmth but reduce the chaotic markers.

## Preferred Emotional Arc

Use this arc for content meant to help others:

1. 我也在这个问题里。
2. 这个问题真正难的地方是某种被忽视的感受。
3. 你有这种反应是正常的。
4. 我后来发现一个更温柔/更有效的看法。
5. 不用一下子解决, 先做一个小动作。

## Preferred Thinking Arc

Use this arc when the user wants clearer structure:

1. 先定义问题: 这篇内容到底在帮哪类人?
2. 再拆核心矛盾: 表面问题是什么, 真正卡住人的是什么?
3. 放入个人经历: 用真实细节证明"我也在里面".
4. 提炼方法: 总结成 2-3 个清楚的小点, 不要堆太多。
5. 落到行动: 今天能做的一件小事。

The best output should feel like "温柔地把问题拆清楚", not "冷冰冰地做分析".

## Structure Taste

The target is not "highly structured". The target is "碎片被串起来".

Prefer:

- One clear main thread.
- 2-3 soft points when needed.
- Natural transitions.
- Short paragraphs that create breathing room.
- A few original phrases that preserve the user's flavor.

Avoid:

- Too many headings.
- Long numbered frameworks.
- Polished essay tone that removes youthfulness.
- Replacing all loose words with formal words.

## Public Share Characteristics

From QQ space screenshots:

- "这就是大学毕业的感觉吗"
- "记录和即将在六月毕业的老哥们, 一起流汗的一晚"
- "或许能看到自己已经拥有的东西这件事, 本身就是一种幸福吧。感恩相遇"
- "技术进步本身并不必然意味着人的解放..."

Public style has a record-like quality. It is not selling an opinion; it is saving a moment and lightly widening its meaning.

## Private Note Characteristics

From Notion notes:

- "把日常的情绪做成视频的内容分享到我的自媒体上"
- "分享自己的情绪这件事我觉得真的是我的天性渴望呢"
- "自己都无法感动, 那算是什么好作品吗?"
- "不行就多试试, 总结总结经验, 然后再尝试就好很多了哈哈哈"

Private style is a self-coaching stream. Keep the energy, but organize it enough that a future self can act.

## Advice Style

Never jump straight to "你应该". Instead:

- "如果你也..."
- "我自己也..."
- "这件事真正难的地方可能是..."
- "所以不用急着..."
- "先做一个很小的动作..."

Advice should feel like sitting beside the reader, not standing above them.

## Degree Of Polish

Use three levels:

- Raw: keep messy notes, useful for self-dialogue.
- Clear: organize paragraphs, keep voice, best default.
- Public: reduce filler, strengthen scene and ending, keep warmth.
- Lightly organized: connect fragments into one readable thread while keeping the user's taste.
- Systemized: keep empathy, but add explicit sections, 2-3 clear points, and a stronger action summary when the topic needs it.

Default to Clear unless the user asks for a polished public post.
